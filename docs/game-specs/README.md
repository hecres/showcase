# ゲーム仕様

開発中ゲームの仕様書からの抜粋です。<br/>
仕様体系の根幹を示す3点と、記法を実例で確認できる代表仕様を掲載しています。

## 仕様書を整備する目的

個人開発でも、仕様書は自分の考えを言語化し、実装前に曖昧な点を明確にするために作成しています。<br/>
また、AI に実装や設計補助を依頼する際の入力資料としても利用し、意図や判断基準を伝えやすくしています。

## 非公開資料について
