# DDD 用語集

| 英語 | 日本語 | 説明 | 使用例 |
|------|--------|------|------|
| Entity | エンティティ | 識別子で同一性を判定するドメインオブジェクト。状態は変化しうる | — |
| ValueObject | 値オブジェクト | 属性の同値で同一性を判定する不変オブジェクト | — |
| Aggregate | 集約 | 整合性境界として一つの単位で扱われるエンティティと値オブジェクトの集合。外部からのアクセスは集約ルート経由のみ | — |
| Service | ドメインサービス | 特定のエンティティに属さないドメインロジックの担い手 | `PieceDamageCalculationService` |
| Repository | リポジトリ | 永続化方式を隠蔽してドメインオブジェクトの取得・保存を担う抽象 | `IPieceRepository` |
