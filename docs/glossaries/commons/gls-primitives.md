# プリミティブ用語集

| 英語 | 日本語 | 説明 | 使用例 |
|------|--------|------|------|
| Angle | 角度 | 度数やラジアンで表す回転量 | `LaunchAngle` |
| Direction | 方向 | 向きや進行方向 | `MoveDirection` |
| Id | 識別子 | オブジェクトを一意に識別する値 | `DataId`, `ChargeVfxId` |
| Index | インデックス | 配列やリスト内の順序位置 | `SlotIndex` |
| Position | 座標 | 空間上の位置 | `SpawnPosition` |
| Range | 範囲 | 値の上下限や距離の範囲 | `AttackRange` |
| Rotation | 回転 | オブジェクトの回転状態 | `InitialRotation` |
| Scale | 拡縮 | オブジェクトの拡大・縮小率 | `ModelScale` |
| Type | タイプ | 種別・分類 | `AlignmentType` |
| Velocity | 速度 | 移動の速さと方向 | `ProjectileVelocity` |
