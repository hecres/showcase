# エラー用語集

| 英語 | 日本語 | 説明 | 使用例 |
|------|--------|------|------|
| ErrorCode | エラーコード | エラータイプ + エラー番号の複合表記。エラーの一意識別子 | `NW-001` |
| ErrorType | エラータイプ | エラーの発生箇所を大まかに分類する略称 | `NW`（ネットワーク） |
| ErrorNumber | エラー番号 | エラータイプ内で個別のエラーを識別する番号 | `001` |
| UncaughtException | 未捕捉例外 | 処理されずに上位へ伝播した例外 | — |
