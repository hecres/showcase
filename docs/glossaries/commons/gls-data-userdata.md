# ユーザーデータ用語集

プレイヤーの進行・所持情報に関する汎用用語。

| 英語 | 日本語 | 説明 | 使用例 |
|------|--------|------|------|
| UserStateData | ユーザー状態データ | プレイヤー状態データの大カテゴリ。ProgressData・InventoryData・Wallet・LoadoutStateData等の総称 | — |
| ProgressData | 進行データ | クエストやチャプターなどの進行状態 | `ChapterProgressData`, `QuestProgressData` |
| InventoryData | インベントリ | アイテムやカードなどの所持情報 | `CardInventoryData`, `GearInventoryData` |
| WalletData | ウォレット | ゲーム内通貨の保有情報 | — |
| LoadoutStateData | 出撃時状態データ | 前回出撃時の状態（選択デッキ等）を引き継ぐデータ | — |
