# 演出用語集

| 英語 | 日本語 | 説明 | 使用例 |
|------|--------|------|------|
| Marker | 位置マーカー | 空間上の基準位置を示す目印 | `PieceAvatarBodyPointMarker` |
| Performer | 演出実行 | エフェクトやアニメーション等の演出の再生・制御の担い手 | `WorldVfxParticleSystemPerformer` |
| Socket | 装着ポイント | 武器やアクセサリ等のオブジェクト装着位置 | `PieceAvatarWeaponSocket` |
| Spawner | 生成器 | エフェクトやオブジェクトの動的生成の担い手 | `WorldVfxSpawner` |
