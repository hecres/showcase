# HecTalk 用語集

`Assets/Hecres/Frameworks/HecTalk/` の会話用語。

| 英語 | 日本語 | 説明 | 使用例 |
|------|--------|------|------|
| TalkSession | 会話セッション | 1回の会話の開始〜終了を管理するライフサイクルオブジェクト | — |
| Talker | 話者 | 会話システムにおける話者（マスターデータとして登録されるキャラクター） | — |
| Participant | 参加者 | 発話に参加している話者（位置・感情・演出の状態を持つ、最大3名） | — |
| Utterer | 発話者 | 発話において現在発話中の参加者 | — |
| Utterance | 発話 | 会話における1文（地の文を含む） | — |
