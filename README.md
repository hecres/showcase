## 技術スタック概要

Unity 6 / C# / .NET Standard 2.1 をベースに開発しています。<br/>
各項目の採用範囲・選定理由・設計判断の詳細は [技術設計資料](docs/game-design/README.md) をご参照ください。

### 採用ライブラリ

- VContainer（DI）
- R3（Reactive）
- UniTask（非同期）

### 採用開発ツール

- Claude Code（AI コーディングエージェント）
- Codex（AI コーディングエージェント）
- uLoopMCP（AI 駆動の Unity 開発自動化）

### 採用アーキテクチャ

- クリーンアーキテクチャ
- DDD
- MVRP（Model-View-Reactive Presenter）
