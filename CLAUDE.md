# Claude AI Resource Layout

このリポジトリでは、Claude Code 向けの AI 開発支援リソースを公開サンプルとして直接配置しています。

Codex 向け資産と同じ品質観点を扱えるよう、対応するスキル・専門レビュアー・参照ルールを `.claude/` 配下にも配置しています。

## 配置

| パス | 内容 |
|---|---|
| `.claude/skills/` | Claude Code 向けスキル。実装、コメント整備、レビュー、ナレッジ改善、コミット、PR 作成の手順を配置 |
| `.claude/agents/` | Claude Code 向け専門レビュアー定義 |
| `.claude/rules/` | スキルから参照する補助ルール |
| `.claude/shared-references/` | 複数スキルから参照する共通ルール |

## 掲載範囲

公開サンプルでは、AI 活用の品質担保が伝わるものに絞って掲載しています。

- C# 実装と実装後レビュー
- XML ドキュメントコメント整備
- 専門レビュアーによる分担レビュー
- 想定と異なる成果物をルール・スキルへ戻す改善フロー
- コミットメッセージと PR 本文の作成手順

## ファイル配置の原則

新規ファイルを追加する際は、以下の原則に従います。

- スキル固有の参照ルールは、対象スキルの `references/` に配置する
- 複数スキルで共有する参照ルールは、`.claude/shared-references/` に配置する
- Claude Code 専用の補助ルールは、`.claude/rules/` に配置する
- レビュアー定義は `.claude/agents/` に配置し、スキルから呼び出す
