# 汎用命名規則

## クラス接尾辞

| 接尾辞 | 意味 | 例 |
|--------|------|-----|
| Base | 基底クラス | `MvrpPresenterBase` |
| Manager | 管理クラス | `SceneManager`, `VfxPoolManager` |
| Proxy | プロキシパターン | `ServiceProxy` |
| Ui | UI表示コンポーネント | `CardUi`, `WalletUi` |

## 命名パターン

| パターン | 意味 | 例 |
|----------|------|-----|
| `XxxFinderUi` | スクロール対応リストUI | `ItemFinderUi`, `StorageFinderUi` |
