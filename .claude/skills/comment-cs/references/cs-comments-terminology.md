# 用語統一規則

コメント記述時の表記ゆれを防止するためのルール。

## 送り仮名の統一

送り仮名は **あり形** で統一する。

| 正 | 誤（使用禁止） |
|----|--------------|
| 切り替え / 切り替える | 切替 / 切替える |
| 呼び出し / 呼び出す | 呼出し / 呼出す |
| 取り出し / 取り出す | 取出し / 取出す |
| 受け取り / 受け取る | 受取 / 受取る |
| 取り消し / 取り消す | 取消 / 取消す |
| 読み取り / 読み取る | 読取 / 読取る |
| 組み合わせ | 組合せ |

## 用語の表記統一

| 正 | 誤（使用禁止） | 備考 |
|----|--------------|------|
| 列挙体 | 列挙型 | C# の型システム上は「列挙型」も正しいが、「列挙体」に統一 |
| 設定します | セットします | カタカナ表記を避け、日本語動詞に統一 |
| 一覧 | リスト | データ構造 `List<T>` 自体を指す場合も「一覧」を使用 |
| 初期化 | イニシャライズ | カタカナ表記を避ける |
