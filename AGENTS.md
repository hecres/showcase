# Codex AI Resource Layout

このリポジトリでは、Codex 向けの AI 開発支援リソースを公開サンプルとして直接配置しています。

この showcase では閲覧しやすさを優先し、必要なスキル・エージェント・参照ルールをリポジトリ内にコミットしています。

## 配置

| パス | 内容 |
|---|---|
| `.agents/skills/` | Codex 向けスキル。実装、コメント整備、レビュー、ナレッジ改善、コミット、PR 作成の手順を配置 |
| `.agents/rules/` | スキルから参照する補助ルール |
| `.agents/skills/shared-references/` | 複数スキルから参照する共通ルール |
| `.codex/agents/` | Codex 向け専門レビュアー定義 |
| `.codex/hooks.json` | Codex hook 設定 |
| `.codex/hooks/` | Codex hook から呼び出す補助スクリプト |

## 掲載範囲

公開サンプルでは、AI 活用の品質担保が伝わるものに絞って掲載しています。

- C# 実装と実装後レビュー
- XML ドキュメントコメント整備
- 専門レビュアーによる分担レビュー
- 想定と異なる成果物をルール・スキルへ戻す改善フロー
- コミットメッセージと PR 本文の作成手順

以下のような、公開時に意図が伝わりにくい運用スキルは掲載対象外です。

- 時刻変更コミット
- 承認省略コミット
- PR 作成からマージまでの一括実行

## ファイル配置の原則

新規ファイルを追加する際は、以下の原則に従います。

- スキル固有の参照ルールは、対象スキルの `references/` に配置する
- 複数スキルで共有する参照ルールは、`.agents/skills/shared-references/` に配置する
- Codex 専用の補助ルールは、`.agents/rules/` に配置する
- レビュアー定義は `.codex/agents/` に配置し、スキルから呼び出す
