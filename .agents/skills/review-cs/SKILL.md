# C# 成果物レビュー

C# コードの成果物を複数の専門レビュアーエージェントで並列レビューし、結果を統合して報告する。

## レビュアーエージェント

以下のエージェントをサブエージェントとして並列実行する。

| エージェント | 観点 |
|-------------|------|
| `cs-reviewer-style` | コーディングスタイル（書式・命名・慣例） |
| `cs-reviewer-style-file-layout` | ファイルレイアウト（メンバー定義順序） |
| `cs-reviewer-comment` | XMLドキュメントコメント・用語 |

## 手順

- [ ] 1. レビュー対象のファイル・差分を特定する
- [ ] 2. 各レビュアーエージェントに対象ファイル一覧を渡し、サブエージェントとして並列起動する
- [ ] 3. 全サブエージェントの完了を待つ
- [ ] 4. 結果を統合し、違反箇所をファイルパス・行番号付きで報告する
- [ ] 5. セルフレビューを実施する

## セルフレビュー

- [ ] 全レビュアーエージェントが起動され、結果が返却されているか
- [ ] 報告漏れがないか（各エージェントの指摘を統合結果に反映しているか）

## サブスキルとして呼び出された場合

他スキル（`/implement-cs` 等）から呼び出された場合、セルフレビュー完了後に **停止せず呼び出し元スキルの次の手順に即座に進む。** ユーザー入力を待たないこと。
