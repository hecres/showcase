# 貢献タイプ分類

PRタイトルに付与するアイコンで、実装の主体と承認プロセスを示す。

## タイプ一覧

| アイコン | タイプ | 判定基準 |
|---------|--------|----------|
| 🤖 | AI自律 | AIが実装し、人間の承認なしにコミットされた |
| 🤝 | AI協業 | AIが実装し、人間がレビュー・承認してコミットされた |
| 👤 | 人間実装 | 人間が実装した |

## 判定フロー

### 🤖 AI自律

以下の **両方** を満たす場合:

- AIがコード変更を実施した
- コミット承認スキップが指示されており、人間の承認なしにコミットが実行された

### 🤝 AI協業

以下の **いずれか** に該当する場合:

- AIが実装し、人間がレビュー・確認したうえで承認した
- 人間の指示に基づきAIが対話的に実装を進めた
- AIの提案を人間が確認し、問題なしと判断してコミットを指示した

### 👤 人間実装

以下の **いずれか** に該当する場合:

- 人間がコードを実装した（AIは分析・相談・調査のみ）
- 人間が実装した後、コミット操作のみをスキル経由で行った
- AIは方針検討や情報提供のみで、実装は人間が行った

## 判定に迷う場合

タイプが混在する場合や判定が難しい場合は、🤝（AI協業）を採用する。

明らかに 🤖 または 👤 に該当する場合のみ、それぞれのタイプを使用する。
