# PR本文テンプレート

このファイルの内容は **PR本文に挿入する雛形そのもの** である。<br/>
書き方の指針・例示は `references/pr-visual-style.md` を参照する。<br/>
このファイル本文にある見出し以外のセクション名（「視覚化指針」「パターン例」等）をPR本文に持ち込まない。

---

## 雛形

下のコードブロック内のMarkdownをPR本文として使用する。<br/>
`{...}` を実内容で置換する。該当なしのセクションは見出しごと削除する。

```markdown
## 概要

{1〜2文で変更の目的を述べる}

## 変更内容

{表またはMermaid図で構造的に提示する。詳細は references/pr-visual-style.md を参照}

## 備考

{この差分に対するレビュー時の注意点のみ。「別途対応」「本PRに含めず」等、差分に現れない作業への言及は禁止。なければセクションごと削除}

---

🤖 AI: {AIの担当内容} / 👤 Human: {人間の担当内容}
```
