# 汎用メソッド テンプレート

汎用メソッドの固定コメント。改変せずそのまま使用すること。

| メソッド | summary |
|----------|---------|
| `Initialize` / `InitializeAsync` | 初期化します。 |
| `Dispose` | リソースを解放します。 |
| `InitializeStaticFields` | 静的フィールドを初期化します。 |
| `InstallBindings` | 依存関係をバインドします。 |
| `InstantiateAndBind` | このクラスのインスタンスを生成し、コンテナへバインドします。 |
