# Unity イベントメソッド テンプレート

MonoBehaviour のライフサイクルメソッドには以下の固定コメントを使用する。
これらのテンプレートは改変せずそのまま使用すること。

## ライフサイクルメソッド

| メソッド | summary |
|----------|---------|
| `Awake` | コンポーネントを初期化します。 |
| `Start` | コンポーネントを初期化します。 |
| `Update` | フレームごとに更新処理を呼び出します。 |
| `FixedUpdate` | 一定時間ごとに物理演算の更新を行います。 |
| `LateUpdate` | 全てのUpdateの完了後に更新処理を行います。 |
| `OnEnable` | コンポーネントが有効になった時に呼び出されます。 |
| `OnDisable` | コンポーネントが無効になった時に呼び出されます。 |
| `OnDestroy` | コンポーネントが破棄された時に呼び出されます。 |
| `OnValidate` | Editor上で設定値を検証します。 |

## 物理コールバック

| メソッド | summary |
|----------|---------|
| `OnTriggerEnter` | トリガーに他のコライダーが侵入した時に呼び出されます。 |
| `OnTriggerExit` | トリガーから他のコライダーが抜けた時に呼び出されます。 |
| `OnTriggerStay` | トリガー内にコライダーが留まっている間、フレームごとに実行されます。 |
| `OnCollisionEnter` | 他のコライダーと衝突した時に呼び出されます。 |
| `OnCollisionExit` | 他のコライダーとの衝突が終了した時に呼び出されます。 |
| `OnCollisionStay` | 衝突し続けている間、フレームごとに実行されます。 |

## 使用例

```csharp
/// <summary>
/// コンポーネントを初期化します。
/// </summary>

{
    // ...
}

/// <summary>
/// フレームごとに更新処理を呼び出します。
/// </summary>
private void Update()
{
    // ...
}
```
